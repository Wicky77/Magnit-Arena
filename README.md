Игра: «Магнит-Арена»
Суть: Вы управляете зеленым квадратом (магнитом) на арене. Вокруг — ящики, враги, ямы и целевые зоны. Используя магнит, либо сбросить всех красных врагов в ямы, либо собрать все коричневые ящики в зеленую зону.

1 неделя:
Разработка ядра модели (Project `Model`). Классы `Player`, `Box`, `Enemy`, `World`. Реализация методов `Move()`, `ApplyMagnetForce()` (векторная математика). Написание модульных тестов (Project `Tests`): проверка изменения координат, проверка направления силы притяжения/отталкивания. В модели нет зависимостей от Windows Forms.

2 неделя:
Подключение View и Controller. Класс `GameForm` (наследник Form). Отрисовка объектов через `Graphics.FillRectangle` в методе `OnPaint`. Обработка ввода `KeyDown` (стрелки, Q, E). Реализация базовой физики: столкновения (`Rectangle.IntersectsWith`), трение (замедление скорости каждый кадр). Тесты на корректность отрисовки координат.

3 неделя:
Логика уровней и победы. Хранение карт в виде массивов `int[,]` (стены, ямы, зоны). Условия: «Все ящики в зоне» или «Враг в яме». Реализация состояний игры (Playing, Won, Lost). Написание тестов на логику победы (сценарий: ящик в зоне → уровень пройден). Сборка работающего прототипа с 3 уровнями.

4 неделя:
Интерфейс и навигация. Главное меню (Form `MainMenu`): кнопки «Старт», «Выход». Экран паузы (overlay на `GameForm`): вызов по `Esc`. Экраны «Победа»/«Поражение» с кнопкой «Рестарт». Текстовые подсказки на экране управления («Q — толкать»). Проверка перехода между экранами без перезапуска процесса.

5 неделя:
Полировка. Визуальные эффекты: линии магнита при нажатии Q/E, мигание объектов при ударе. Звуки через `SoundPlayer` (шаги, удар, победа). Балансировка силы магнита и скорости. Финальный прогон всех модульных тестов. Убедиться, что игра проходится за 3-5 минут.

6 неделя:
Резерв и подготовка к сдаче. Буфер времени на случай непредвиденных багов. Рефакторинг кода (имена переменных, удаление лишнего). Подготовка краткого рассказа для преподавателя (архитектура MVC, где тесты). Репетиция демонстрации (тайминг, сценарий показа).
